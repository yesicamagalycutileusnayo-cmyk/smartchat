💡 El Proyecto

¿Qué es SmartChat?

SmartChat es un sistema web de apoyo emocional inteligente desarrollado como proyecto académico de Ingeniería de Sistemas. Combina interfaces modernas con simulación de inteligencia artificial para brindar apoyo emocional accesible y sin barreras.

El sistema responde a estados emocionales como tristeza, estrés, ansiedad y falta de motivación, ofreciendo respuestas empáticas y técnicas de bienestar basadas en principios de psicología cognitivo-conductual.

🧠

Respuestas contextuales

Detecta el estado emocional y responde apropiadamente.

🔒

Privacidad garantizada

Conversaciones locales, sin almacenamiento en servidores.

📱

Acceso desde cualquier dispositivo

Diseño responsive para móvil, tablet y escritorio.

🤖

Tecnología al servicio del bienestar

Desarrollado con HTML, CSS y JavaScript puro. Sin dependencias externas. 100% frontend, rápido y accesible.

HTML5 CSS3 JavaScript localStorage
